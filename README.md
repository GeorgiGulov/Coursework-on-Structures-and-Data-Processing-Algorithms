# Coursework-on-Structures-and-Data-Processing-Algorithms
Development of an information system for the subject area - "Customer service of a mobile operator" using the given data structures and algorithms:
  − hashing method – Private hashing with double hashing;
  − sorting method – Extraction;
  − list type – Cyclic bidirectional;
  − tree traversal method – Direct;
  − word search algorithm in the text – Direct.
